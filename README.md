# Framework Forge

Turns messy ideas into a versioned framework package with memory, critique, and export.

## Product Goal
Take a rough idea and turn it into a structured, reusable framework package with:
- the real objective
- hidden assumptions
- key constraints
- framework structure
- execution plan
- weak points
- best next prompts
- saved memory for future refinement

## Core Principle
Do not let the model be the framework.
Make the model produce the framework.

## Initial Scope
This repo starts as the home for the product definition, package structure, and versioned system design before any implementation work begins.
